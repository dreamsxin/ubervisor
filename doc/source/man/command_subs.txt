==============
ubervisor-subs
==============

Synopsis
========

``ubervisor`` *subs* *ident*

Description
===========

Subscribe to server notifications.

Supported values for *ident* are:

- 1 (server log)
- 2 (group status update)
- 4 (group configuration updates)

========
